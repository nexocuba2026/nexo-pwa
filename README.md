# Hostal Cienfuegos - Sitio Web

Sitio web moderno y minimalista para un hostal en Cienfuegos, Cuba.

## 🚀 Características

- Diseño responsive y moderno
- Sistema de reservas integrado
- Solicitud de servicios de transportación
- Integración con WhatsApp
- Formulario de newsletter
- Galería de imágenes
- Mapa de ubicación
- Optimizado para SEO

## 📋 Requisitos Previos

- Navegador web moderno
- Servidor web (opcional para desarrollo local)
- Cuenta de GitHub para despliegue
